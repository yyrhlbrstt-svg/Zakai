# GO-LIVE — הדלקת זכאי ב-10 דקות (למייסד)

הקוד מוכן ב-100%: 24 כלים, גביית עמלה מאומתת, לולאה ויראלית, אבטחה, 188 טסטים.
מה שנשאר זה **להדליק** — ארבעה צעדים, לפי סדר. עשה אותם וזה חי מול משתמשים.

---

## ✅ צעד 1 — פרוס לפרודקשן (30 שניות)
1. Vercel → הפרויקט **zakai** → לשונית **Deployments**.
2. מצא את הפריסה האחרונה של הברנץ' `claude/zakai-consumer-ai-agent-3ucncj` (ירוקה, Ready).
3. לחץ על ⋯ ← **Promote to Production**.
4. פתח `https://zakai-3uxj.vercel.app/he` — זה הלינק לשלוח לאנשים.

> אם main מחובר לאוטו-דיפלוי, אפשר גם פשוט למזג את הברנץ' ל-main.

## ✅ צעד 2 — הדלק את הסוכן החכם, בחינם (2 דקות)
1. `console.groq.com` → הרשמה → **API Keys** → Create (חינם, בלי כרטיס).
2. Vercel → הפרויקט → **Settings → Environment Variables**, הוסף שלושה:
   - `OPENAI_COMPAT_API_KEY` = המפתח מ-Groq
   - `OPENAI_COMPAT_BASE_URL` = `https://api.groq.com/openai/v1`
   - `OPENAI_COMPAT_MODEL` = `llama-3.3-70b-versatile`
3. **Redeploy** (Deployments → האחרונה → ⋯ → Redeploy). עכשiv הסוכן חי ומהיר.

> יש לך קרדיט Gemini? במקום Groq שים `GEMINI_API_KEY` — הקוד מזהה לבד.

## ✅ צעד 3 — סליקה, כשתרצה לגבות עמלה בפועל (ימים)
הצנרת בנויה ומאומתת (PENDING→PAID). כדי לגבות כסף אמיתי:
1. פתח חשבון סליקה ישראלי (PayPlus / Tranzila / Cardcom / Grow) — דורש ישות עסקית.
2. תגיד לי איזה ספק בחרת — אני מחבר את ה-adapter (`src/lib/payments`) מול ה-API שלו + אימות חתימת webhook.
3. Vercel: `PAYMENT_PROVIDER` = שם הספק + המפתחות שלו.

> עד אז: כפתור "שלם עמלה" עובד במצב הדגמה. אין הכנסה אמיתית בלי צעד זה.

## ✅ צעד 4 — 10 המשתמשים הראשונים (השבוע)
1. שלח את הלינק ל-10 אנשים מהמשפחה/חברים בוואטסאפ.
2. בקש מהם לעשות **"מה מגיע לי"** (30 שניות) או להעלות חשבונית.
3. הראשון שמקבל תוצאה אמיתית → זו העדות הראשונה → ממנה מתחילה הוויראליות.
